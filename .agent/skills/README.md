# Gemini CLI Skill Registry

This directory contains the definitions and role descriptions for the Multi-Agent System.

## Core Agents (Software Engineering)
- `team_lead.md`: Orchestration & Architecture.
- `principal_developer.md`: Backend & Core Implementation.
- `analyst.md`: Requirements & Security Analysis.
- `designer.md`: UI/UX & Motion.
- `qa_engineer.md`: Testing & Quality Assurance.

## Game Development Extension (xGame)
Specialized roles for full-cycle game production.

### Development
- `game_architect.md`: Engine selection, system architecture, technical leadership.
- `gameplay_developer.md`: Mechanics implementation (C#, C++, Blueprints).

### Creative
- `art_director.md`: Visual style, 3D/2D asset pipeline, level design.

### Management & Growth
- `project_manager.md`: Agile/Scrum, GDD maintenance, roadmap planning.
- `marketing_specialist.md`: ASO, community management, launch strategy.
